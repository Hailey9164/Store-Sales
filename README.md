# Store-Sales
CSC/CIS-5-48828/48893  Fall 2023 Program Logic Using C++ - lab 9c - store sales

Dynamic Allocation & File‑Based Sales Analysis

## Overview
This program reads a user‑specified input file containing yearly sales data for Professor Locke’s Seed, Feed, and Fertilizer Stores. It dynamically allocates an array to store the sales figures, processes the values, and outputs formatted statistics including totals, averages, and the highest/lowest‑grossing stores.
The assignment demonstrates:
• 	File input handling
• 	Dynamic memory allocation using  and 
• 	Array processing
• 	Basic statistical computation
• 	Formatted console output
